# Post Provision Checklist

These are manual steps that cannot be automated after running `./provision`.

## Finder

- [ ] Manually reorder Finder sidebar favorites:
       Drag to reorder: Airdrop, Recents, Applications, Home, Desktop, Developer, Documents, Music, Downloads

## Menu Bar

- [ ] Hide unwanted menu bar icons manually (or use Bartender if installed)
- [ ] Reorder menu bar icons by holding `Command` and dragging

## Raycast

- [ ] Set `Command + Space` to launch Raycast
       Raycast → Settings → Hotkey → Set to `Command + Space`
