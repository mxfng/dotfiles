# Workflow Shortcuts

## Quick Launch
| Shortcut | Action |
|----------|--------|
| ⌥ + space | Open terminal (WezTerm) |
| shift + ⌥ + return | Open web browser (Arc) |

# Window Management Shortcuts

## Focus Windows
| Shortcut | Action |
|----------|--------|
| ⌥ + h/j/k/l | Focus window in direction (vim keys) |

## Move Windows
| Shortcut | Action |
|----------|--------|
| shift + ⌥ + h/j/k/l | Swap window positions |
| shift + ⌘ + h/j/k/l | Move window in direction |

## Spaces
| Shortcut | Action |
|----------|--------|
| ⌘ + ⌥ + 1-9 | Focus space |
| shift + ⌘ + 1-9 | Move window to space |

## Window States
| Shortcut | Action |
|----------|--------|
| ⌥ + t | Toggle float |
| ⌥ + p | Picture-in-picture |
| ⌥ + f | Fullscreen |
| ⌥ + d | Parent zoom |

## Layout
| Shortcut | Action |
|----------|--------|
| ⌥ + e | Toggle split |
| ⌥ + r | Rotate tree |

> Note: all commands for yabai at ~/.skhdrc

# Tmux Shortcuts

> Note: prefix is ctrl+a

## Sessions
| Command | Action |
|---------|--------|
| tmux new -s name | Create new named session |
| tmux ls | List sessions |
| tmux attach -t name | Attach to named session |
| prefix + d | Detach from session |

## Windows (Tabs)
| Shortcut | Action |
|----------|--------|
| prefix + c | Create new window |
| prefix + n | Next window |
| prefix + p | Previous window |
| prefix + w | List windows |
| prefix + , | Rename window |
| prefix + & | Kill window |

## Panes (Splits)
| Shortcut | Action |
|----------|--------|
| prefix + " | Split vertically |
| prefix + % | Split horizontally |
| prefix + x | Kill pane |
| prefix + arrows | Navigate between panes |
| prefix + z | Toggle pane zoom |

## Misc
| Shortcut | Action |
|----------|--------|
| prefix + t | Show the time |
